# portfolio-theme

A minimal portfolio built with HTML, CSS, and JavaScript.

## Setup

1. Clone the repo
2. Replace `Your Name` with your name across all files
3. Replace `xx` with your initials
4. Update your social links, bio, and projects
5. Add blog posts to `content/blog/posts/` and register them in `content/blog/posts.json`

## Blog

Posts are written in Markdown. Each post needs an entry in `posts.json` with an `id`, `title`, `date`, `slug`, and `excerpt`. The slug must match the filename.
